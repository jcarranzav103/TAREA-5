DOM (Document Object Model)

  • DOM
DOM

Se refiere a la representación del documento HTML o XML en forma de un árbol de objetos JavaScript que podemos manipular mediante código JavaScript. En otras palabras, el DOM es una interfaz que permite acceder y manipular los elementos de un documento HTML o XML desde JavaScript.

El DOM se organiza como una estructura en forma de árbol, donde el nodo raíz es el objeto document que representa el documento web actual. A partir de este nodo raíz, podemos acceder a todos los elementos del documento, como etiquetas HTML, atributos, texto, imágenes, enlaces, etc. Cada elemento del árbol es un objeto JavaScript que tiene propiedades y métodos que podemos utilizar para acceder o modificar su contenido y su apariencia visual.
