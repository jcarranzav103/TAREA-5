Objeto Number

En JavaScript, el objeto Number representa un número en el lenguaje de programación. El objeto Number es una instancia de la clase Number, y proporciona una serie de métodos y propiedades para trabajar con números.



En el primer ejemplo, se crea una instancia del objeto Number con el valor 10 y se asigna a la variable num. Luego, se utilizan algunos de los métodos proporcionados por el objeto Number para trabajar con este número. El método valueOf() devuelve el valor numérico del objeto, el método toFixed() devuelve una cadena que representa el número con una cantidad fija de dígitos decimales, el método toPrecision() devuelve una cadena que representa el número con una cantidad fija de dígitos significativos, y el método toString() devuelve una cadena que representa el número en una base específica (en este caso, base 2).



En el segundo ejemplo, se crea una variable num2 con el valor 5.67. Luego, se utilizan algunos de los métodos proporcionados por el objeto Number que también funcionan con números de tipo primitivo. En este caso, el método toString() devuelve una cadena que representa el número, el método toFixed() devuelve una cadena que representa el número con una cantidad fija de dígitos decimales, y el método toPrecision() devuelve una cadena que representa el número con una cantidad fija de dígitos significativos.

Objeto Math

En JavaScript, el objeto Math es un objeto incorporado que proporciona una serie de métodos y propiedades matemáticas útiles. No es un objeto que se instancia, sino que se puede utilizar directamente desde el objeto Math.

Ejemplo


El objeto Math también proporciona otros métodos y propiedades matemáticas útiles, como abs(), sin(), cos(), tan(), log(), max(), min(), entre otros.


Objeto Date

En JavaScript, el objeto Date es un objeto incorporado que proporciona una forma de trabajar con fechas y horas. El objeto Date se utiliza para crear y manipular fechas y horas, y proporciona una serie de métodos y propiedades útiles para trabajar con ellas.
