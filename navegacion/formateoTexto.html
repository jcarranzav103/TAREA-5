Cadenas literales

En JavaScript, las cadenas literales son una forma de crear cadenas de texto de manera más simple y legible en el código. Las cadenas literales se crean utilizando comillas simples (') o dobles (") alrededor del texto que se desea incluir en la cadena.



En el primer ejemplo, se crean dos variables que contienen cadenas de texto. Luego, se utiliza la concatenación de cadenas (mediante el operador +) para crear una nueva cadena que incluye el nombre completo. En el segundo ejemplo, se utiliza una cadena literal con interpolación de variables (mediante el uso de la sintaxis ${}) para crear la misma cadena de una manera más legible.



Las cadenas literales también pueden contener caracteres especiales, como saltos de línea (\n), tabulaciones (\t), comillas (\" o \'), entre otros.

Objeto String

En JavaScript, el objeto String es un objeto incorporado que proporciona una serie de métodos y propiedades útiles para trabajar con cadenas de texto. Los objetos String son inmutables, lo que significa que una vez que se crea una cadena, no se puede modificar.
